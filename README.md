# Fitness and Fat Loss Guidance Website

## Overview
This is a web-based fitness website developed using HTML, CSS, and JavaScript. The website provides a 7-day fat loss plan, diet recommendations, and exercise video guidance.

## Features
- 7-day fat loss plan
- Diet recommendations
- Exercise video guidance
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
Developed as an academic web development project.
